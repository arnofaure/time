# Time Tracker

A 100% free, single-file time tracking web app — created by [Arno Faure](https://arnofaure.com).

🔗 **[time.arnofaure.com](https://time.arnofaure.com)** · 📝 [Changelog](CHANGELOG.md)

## Features

- Organize your work as **Clients → Projects → Tasks**
- Start/stop **timer** on any task (only one runs at a time)
- **Manual time entry** — type a duration (e.g. `1h30`) or set exact start/end times; edit or delete any past entry from a task's history
- **Weekly calendar view** — see your tracked sessions laid out on a day/hour grid, colored by client, with a live "now" line and weekly totals
- **Hourly rate** — one default rate for the whole app, with an optional per-project override
- **Billable / unbillable** tasks — time still counts toward totals, cost is excluded when marked unbillable
- **Client colors** — auto-assigned, click to cycle; used as the client card outline in the list view and as the block color in the calendar view
- Everything **autosaves** to your browser's local storage — nothing to click, nothing lost on reload
- **Export / Import** your data as JSON for backup or transfer
- Runs entirely in your browser — no account, no server, no tracking

## Usage

Just open `index.html` in a browser, or visit [time.arnofaure.com](https://time.arnofaure.com).

## How to use it

1. Add a **client**, then a **project** inside it, then a **task** inside that project.
2. Click the green ▶ on a task to start the timer, click again (■) to stop it.
3. Use the **+** icon on a task to log time manually — by duration or by exact start/end time.
4. Use the clock icon on a task to view, edit or delete past time entries.
5. Set your **default hourly rate** from the logo menu (top left); override it per project if needed.
6. Mark a task **unbillable** with the `$` icon if its time shouldn't be charged.
7. Switch to the **Calendar** view (top bar) to see your week laid out visually.
8. Export your data regularly from the logo menu — it's your only backup, since everything lives in this browser only.
